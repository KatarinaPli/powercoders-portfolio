Personal Portfolio

Are you often struggling with the unpredictable weather and dress for guesswork? My one of the first project in web development was the weather forecast website. It was created using HTML, CSS, Javascript and Bootstrap. Also, using JavaScript, I connected the user's search to an API, thanks to which you can see the actual time, temperature, date and weather description for 7 days in the city that you enter in the search. You can see the creation process on my GitHub page in the public domain, and there is also a link at the bottom of my site.
